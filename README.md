## Identity & Security

- [**cactl**](https://github.com/seb07-cloud/cactl) - The deploy framework for Entra Conditional Access policies. Plan/apply workflows, Git-native versioning, per-policy semver, drift detection & rollback.
- [**pim-tui**](https://github.com/seb07-cloud/pim-tui) - Terminal UI for Azure Privileged Identity Management. Activate/deactivate Entra ID roles, PIM groups, and Azure RBAC roles.
- [**entrasnapshot.com**](https://github.com/seb07-cloud/entrasnapshot.com) - Monitor, detect, and remediate configuration drift in Microsoft Entra tenants. Baseline capture, change tracking across 18 categories, side-by-side diffs.

## Cloud & Infrastructure

- [**avd-health-monitor**](https://github.com/seb07-cloud/avd-health-monitor) - Windows system tray app that tracks Azure Virtual Desktop endpoint connectivity in real time. Latency testing, live graphs, toast alerts.

- [**msgraph-terraform**](https://github.com/seb07-cloud/msgraph-terraform) - Blog companion: managing Microsoft Graph resources with Terraform.
